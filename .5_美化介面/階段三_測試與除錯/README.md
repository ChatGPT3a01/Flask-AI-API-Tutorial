# 2.5 美化介面 - 階段三：測試與除錯

## 學習目標
- 測試各種使用情境
- 使用瀏覽器開發者工具除錯
- 完成功能完整的 AI 對話應用

## 這是完整可運作的版本

包含：
- 美觀的對話介面
- 完整的前後端互動
- 錯誤處理
- 載入狀態提示

## 執行方式

```bash
cd AI_Projects
venv\Scripts\activate
cd "2.5_美化介面/階段三_測試與除錯"
python app.py
```

開啟瀏覽器 http://localhost:5000/

## 測試項目

1. **連續發送多則訊息** - 觀察對話記錄累積
2. **發送空訊息** - 應該被阻止
3. **發送很長的訊息** - 觀察換行效果
4. **網路錯誤** - 停止 Flask 後發送訊息

## 使用開發者工具

按 F12 開啟開發者工具：
- Network 分頁：查看 API 請求
- Console 分頁：查看錯誤訊息
